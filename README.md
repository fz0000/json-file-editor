# json-file-editor

if your json files has the same structure, you can put them in the same folder and edit them together.

Supported json file structure:

```json
{
  "key1": "value1",
  "key2": "value2",
  "key3": [
    "value3.1",
    "value3.2",
    "value3.3"
  ],
  "keyN": "valueN"
}
```
